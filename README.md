# Amy Hughes Portfolio Website

Portfolio website for Amy Hughes - Actor & Comedian

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Environment Variables

Copy `.env.local.example` to `.env.local` and fill in all required API keys and credentials.

## Key Features

- **Google Drive Integration**: Media galleries, showreels, voice demos, resume
- **Dynamic Bio**: Automatically scales text to fit container
- **Contact Form**: Sends emails via Resend API
- **Latest Shows**: Automatically fetches and displays upcoming events

## Tech Stack

- Next.js 14
- TypeScript
- Google Drive API
- Resend (Email)
